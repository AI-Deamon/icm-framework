# Browser Flow: Claude Chrome Extension

**Source:** Skool: Implementation Playbooks / Module 2 / 2.1-2.4 Browser Flow

## Core Idea

The Claude Chrome extension turns your browser into an AI coworker that can read any page, extract structured data, learn your workflows, and automate repetitive browser tasks — all from the sidebar. It's the practical interface layer between the folder system and the web.

## Why It Matters

- 2.2 Pull Structured Data: extract tables, product info, contact details from any website into usable formats using natural language in the sidebar.
- 2.3 Teach Claude Your Workflow: record browser actions (with microphone) and Claude replays them as repeatable shortcuts triggered by a single command.
- 2.4 Inbox and Scheduling: deep walkthrough of using Claude sidebar inside Gmail to triage email and manage calendar in minutes instead of hours — with safety guardrails.
- The browser is where most people live (email, research, SaaS tools). The extension brings the folder system's context-aware AI to that environment. Same principles (context files, routing), different surface.

## How I'll Apply It

Set up the Claude Chrome extension for: (1) extracting research into CONTEXT.md files when exploring competitor sites, (2) recording my Gmail triage workflow as a repeatable shortcut, (3) using structured data extraction to populate project briefs from client websites.

## Cross-Reference

Connects to the scraping/tooling layer of the folder system — the browser extension is a skill that feeds context files. The structured data output goes directly into my workspaces.
